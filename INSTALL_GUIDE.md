# 安装和使用指南

## 快速安装方法

### 方法1: GitHub Actions自动构建（推荐）
1. 将整个项目上传到GitHub
2. GitHub Actions会自动构建APK
3. 在Actions页面下载构建好的APK
4. 在Android设备上安装

### 方法2: 本地构建
1. 使用Android Studio打开项目
2. 点击"Build" > "Build Bundle(s) / APK(s)" > "Build APK(s)"
3. 等待构建完成
4. 在`app/build/outputs/apk/debug/`找到APK文件

### 方法3: 命令行构建
```bash
# 克隆项目
git clone https://github.com/yourusername/WeChatEnhancer_iOS_UI.git

# 进入项目目录
cd WeChatEnhancer_iOS_UI

# 构建Debug版本
./gradlew assembleDebug

# 构建Release版本
./gradlew assembleRelease
```

## 安装步骤

### 1. 允许未知来源安装
1. 进入Android设置
2. 找到"安全"或"隐私"设置
3. 启用"未知来源"或"允许安装未知应用"

### 2. 安装APK
1. 找到下载的APK文件
2. 点击安装
3. 按照提示完成安装

### 3. 首次运行
1. 打开"微信增强模块"应用
2. 授予必要的权限
3. 查看功能介绍
4. 根据需要启用功能

## 功能配置

### 基本功能开关
1. **朋友圈防撤回**: 防止朋友圈内容被撤回
2. **朋友圈去广告**: 移除朋友圈中的广告内容
3. **消息防撤回**: 防止聊天消息被撤回
4. **自动原图**: 发送图片时自动选择原图

### 性能设置
1. **性能模式**: 降低动画效果，减少资源占用
   - 建议在低端设备上启用
   - 可以显著提升流畅度
2. **启用动画**: 控制iOS风格过渡动画
   - 关闭可进一步提升性能
3. **自动更新**: 自动检查新版本

## 使用技巧

### 最佳实践
1. **首次使用**: 建议先启用所有功能进行测试
2. **性能优化**: 如果感觉卡顿，启用性能模式
3. **电池优化**: 将应用加入电池优化白名单
4. **定期更新**: 开启自动更新获取最新功能

### 故障排除
1. **应用闪退**: 
   - 清除应用数据
   - 重新安装
   - 检查Android版本兼容性

2. **功能不生效**:
   - 确保模块已激活
   - 检查功能开关状态
   - 重启微信应用

3. **性能问题**:
   - 启用性能模式
   - 关闭动画效果
   - 清理设备内存

## 高级配置

> 注意: 本应用仅为UI界面，实际功能需要Xposed模块支持

如果需要实际的功能实现：
1. 安装Xposed框架

## 常见问题

### Q: 需要Root权限吗？
A: 不需要。本应用为普通Android应用，不需要Root权限。

### Q: 需要Xposed框架吗？
A: 本应用界面不需要Xposed，但如果需要实际的功能实现（如防撤回），需要配合Xposed模块。

### Q: 支持哪些Android版本？
A: 支持Android 5.0 (API 21) 及以上版本。

### Q: 占用多少存储空间？
A: 安装包约5-10MB，运行时不占用额外存储。

### Q: 耗电吗？
A: 非常省电，仅在用户操作时运行，无后台服务。
